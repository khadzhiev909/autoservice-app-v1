Мы создадим шаблон для отчета RevenueReport.jrxml, который будет включать:

Период (например, даты начала и окончания периода).
Типы услуг.
Количество выполненных услуг.
Доход от каждой услуги.
Общий доход за период.
Пример структуры отчета в шаблоне:

Заголовок: Название отчета и период.
Таблица с колонками:
Дата
Тип услуги
Количество выполненных услуг
Доход по каждой услуге
Общий доход

Этот шаблон имеет следующие части:

Title: Заголовок отчета и период.
Detail: Раздел с информацией о доходах, где будет отображаться дата, тип услуги, количество и доход.
PageFooter: Подпись с датой генерации отчета.
